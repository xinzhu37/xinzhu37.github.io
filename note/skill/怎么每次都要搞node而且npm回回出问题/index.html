    1.Node的版本选择

    对于自己的电脑，并且用来学习知识的话，用最新的就行。如果是运行项目或者在公司办公的话，选择较低版本的（nodejs的正式版本都是偶数2,4,6,8这样命名，测试版奇数1,3,5,7,9）

    2.windows下 node版本管理工具nvm

    下载安装nvm

    点击我下载nvm

    下载 nvm-setup.zip版本，傻瓜式一键安装

    2.常用nvm命令

    cmd命令下请使用管理员模式。

    查看nvm版本信息

    nvm v

    查看可用的node版本

    nvm list available

    查看安装过的node版本

    nvm ls  或者 nvm list

    安装指定版本

    nvm install 版本号   例如：nvm install 14.18.2

    使用某个node版本

    踩坑一

    我一开始使用NVM时无法切换node版本。。。。很尴尬，因为我一开始并不知道要先卸载node，啊啊啊~~~~~，后面卸载了node之后重新安装nvm就可以了

    踩坑二

    当我按照步骤的install我的node版本之后，使用nvm use 8.9.4的时候。。。。。。我竟然乱码。。。。。各种乱。。。。百度了一下很难受，我没有安装在C盘，尴不尴尬，然后还有看到其他回答说命名有空格也会出现这个乱码的问题，注意一下命名规范哈。

    nvm use 14.18.2

    卸载某个node版本

    nvm uninstall 版本号  例如: nvm uninstall 14.18.2
